# Installation

## Core Package

<small>With Yarn</small>

```bash
yarn add @pulse/core
```

<small>With NPM</small>

```bash
npm install --save @pulse/core
```

## React Package

<small>With Yarn</small>

```bash
yarn add @pulse/core @pulse/react
```

<small>With NPM</small>

```bash
npm install --save @pulse/coree @pulse/react
```

## NextJS Package

<small>With Yarn</small>

```bash
yarn add @pulse/core @pulse/react @pulse/next
```

<small>With NPM</small>

```bash
npm install --save @pulse/core @pulse/react @pulse/next
```
