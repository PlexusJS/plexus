## What is it?

Plexus is a SSOT toolkit for building out scaleable reactive applications. In other words, we make state management, api communication, event management, & data manipulation a breeze.

## What does "Plexus" mean?

<p>
	According to a dictionary somewhere, "plexus" means <i>"a network of nerves or vessels in the body"</i>. The way plexus is supposed to work, it
	fits that definition almost perfectly. The tools in this package span the entire project and is interconnected like nerves in the body!
</p>
